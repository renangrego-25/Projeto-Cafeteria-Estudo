# Projeto Cafeteria

Site fictício para uma cafeteria, desenvolvido com o objetivo de praticar 
conceitos de HTML e CSS, criando um layout limpo, organizado e com um 
visual suave e agradável.

# Tecnologias

- HTML5
- CSS3

# O que foi praticado

- Estruturação semântica de página com HTML
- Estilização e layout com CSS (flexbox/grid)
- Paleta de cores e tipografia
- Responsividade
